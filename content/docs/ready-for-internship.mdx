### Understanding Solana SPL Programs

#### SPL Token Program

- **Token Creation**: Mint new tokens with metadata
- **Token Management**: Transfer, burn, freeze, thaw
- **Mint Authority**: Control over token supply
- **Freeze Authority**: Ability to freeze accounts

#### SPL Associated Token Account (ATA)

- **Purpose**: Dedicated account for specific token mint
- **Derivation**: Deterministic address calculation
- **Benefits**: Simplified token management
- **Usage**: Automatic account creation and management

#### SPL Token-2022 Program

- **Enhanced Features**: Transfer fees, interest-bearing tokens
- **Backward Compatibility**: Works with existing SPL tokens
- **Advanced Functionality**: Confidential transfers, metadata
- **Implementation**: Extended token program capabilities

#### Anchor Testing Framework

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use anchor_lang::solana_program::clock::Epoch;
    use anchor_spl::token::{Mint, Token, TokenAccount};
    use anchor_lang::solana_program::system_program;

    #[test]
    fn test_initialize() {
        let program = anchor_lang::solana_program::system_program::ID;
        let (pda, _bump) = Pubkey::find_program_address(
            &[b"test"],
            &program,
        );

        let mut accounts = Initialize {
            base_account: pda,
            user: test_user.pubkey(),
            system_program: system_program::ID,
        };

        let ix = Instruction {
            program_id: program,
            accounts: vec![
                AccountMeta::new(accounts.base_account, false),
                AccountMeta::new(accounts.user, true),
                AccountMeta::new_readonly(accounts.system_program, false),
            ],
            data: vec![],
        };

        let result = process_instruction(&program, &accounts, &ix);
        assert!(result.is_ok());
    }
}
```

### Starting Small Projects

#### Custom Hook for Token-2022

**Objective**: Create reusable React hook for token operations

**Features**:

- Token balance queries
- Transfer operations
- Mint/burn functionality
- Transaction status tracking

**Implementation**:

```typescript
import { useConnection, useWallet } from "@solana/wallet-adapter-react";
import { useMemo, useState, useEffect } from "react";

export const useToken2022 = (mintAddress: string) => {
  const { connection } = useConnection();
  const { publicKey } = useWallet();
  const [balance, setBalance] = useState<number>(0);
  const [loading, setLoading] = useState<boolean>(false);

  const getBalance = async () => {
    if (!publicKey || !mintAddress) return;

    setLoading(true);
    try {
      const tokenAccounts = await connection.getParsedTokenAccountsByOwner(
        publicKey,
        { mint: new PublicKey(mintAddress) }
      );

      if (tokenAccounts.value.length > 0) {
        setBalance(
          tokenAccounts.value[0].account.data.parsed.info.tokenAmount.uiAmount
        );
      }
    } catch (error) {
      console.error("Error fetching balance:", error);
    } finally {
      setLoading(false);
    }
  };

  useEffect(() => {
    getBalance();
  }, [publicKey, mintAddress]);

  return { balance, loading, refreshBalance: getBalance };
};
```
